Cloner le repo :

```bash
git clone https://github.com/daadaamed/cartoonize-image.git
cd cartoonize-image
```

réer un environnement virtuel :
```
python3 -m venv venv
source venv/bin/activate
```
 Installer les dépendances :
```
pip install -r requirements.txt
```
Lancer l’application :
```
python app.py
```
